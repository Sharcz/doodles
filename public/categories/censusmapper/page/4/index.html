Census Mapper

Making StatCan open data more accessible

September 28, 2015 Jens von Bergmann

3 minute read

We are excited to announce CensusMapper public beta launch! The project on mapping census data for Canada is entering the next stage. We are now mapping 3875 different variables from the 2011 census, or any combination of them, across 67215 geographic regions covering all of Canada. We are opening up the Census Mapper to a limited public beta. Limited means that anyone can view maps created in CensusMapper, but we are only opening up the map creation tools to selected beta users.
